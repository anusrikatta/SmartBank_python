# SmartBank – GUI Banking Application (Python)

SmartBank is a graphical banking application built using Python and the Tkinter library. The application provides a simple user interface for performing banking operations such as account creation, login authentication, deposit, withdrawal, and balance checking.

## 🚀 Features

* Graphical user interface (GUI)
* Account registration system
* Secure login with PIN
* Deposit and withdraw money
* Balance inquiry
* Transaction history logging
* Persistent data storage using file handling

## 🛠 Technologies Used

* Python
* Tkinter (GUI)
* Object-Oriented Programming (OOP)
* File Handling
